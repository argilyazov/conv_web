Конфиденциальность

Любые данные, вставленные и преобразованные в csv json, остаются локальными на вашем компьютере. Данные никогда не отправляются на сервер.

О формате данных

Json

JSON расшифровывается как JavaScript Object Notation и стал де-факто компьютерным форматом, доступным для чтения людьми для хранения структурированных данных. От API до файлов конфигурации JSON теперь везде.

Excel

Форматы в Excel отвечают за то, как будут отображаться данные в ячейках. Форматирование позволяет из разнородного набора информации — текста, чисел — сделать структурированную и читаемую таблицу.

markdown

Язык разметки Markdown широко распространён в вебе. На нём пишут readme-файлы и документацию, а его принципы работы используют для оформления сообщений и публикаций в мессенджерах и социальных сетях.

Jira

Плагин импортеров JIRA, который поставляется вместе с JIRA, позволяет импортировать ваши данные из файла с разделителями-запятыми (CSV). CSV-файлы представляют собой текстовые файлы, представляющие табличные данные и поддерживаемые большинством приложений, которые обрабатывают табличные данные
